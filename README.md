# Spotlight

Spotlight — это шаблон сайта [Tailwind Plus](https://tailwindcss.com/plus), созданный с использованием [Tailwind CSS](https://tailwindcss.com) и [Next.js](https://nextjs.org).

## Начало работы

Чтобы начать работу с этим шаблоном, сначала установите npm зависимости:

```bash
npm install
```

Затем создайте файл `.env.local` в корне вашего проекта и установите переменную `NEXT_PUBLIC_SITE_URL` на публичный URL вашего сайта:

```
NEXT_PUBLIC_SITE_URL=https://example.com
```

Далее запустите сервер разработки:

```bash
npm run dev
```

Наконец, откройте [http://localhost:3000](http://localhost:3000) в вашем браузере, чтобы просмотреть веб-сайт.

## Настройка

Вы можете начать редактирование этого шаблона, изменив файлы в папке `/src`. Сайт будет автоматически обновляться при редактировании этих файлов.
